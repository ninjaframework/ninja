Ninja web framework

Full stack web framework for Java.
Fast and productive. Runs on servlet containers - built with Maven.



Happy business. Happy customers.

Ninja will let you finish your next project faster in stunning high quality. Your customers will love what they'll get.

Fun for developers.

Ninja provides everything you need to get productive in an instant. And Ninja is fast. Really fast. Make a change to the code and instantly see the results.





Get started!





Stable technology

Ninja did not reinvent the wheel - instead we are standing on the shoulders of giants. Ninja is built using the most stable technologies in the industry. Servlets, Guice dependency injection and Maven - just to name a few.

Scalability

Ninja uses the share nothing approach. Servers do not share any common session information. That means you can scale Ninja horizontally with ease.

Ready to start

Ninja is a full stack. From quick development to fast testing to easy deployment. And by the way - if you want you can use any servlet container to run your Ninja application.



Pure Open source

Ninja is licensed under Apache 2.0 license. That means the codebase will be open forever. We are welcoming pull requests :)

Powerful build system

Maven is one of the best build and library management systems out there. And Ninja is built for it. Even better: Automating your builds in Jenkins becomes as easy as telling Jenkins your repository location.

First class Java citizen

You get all goodies that make Java great: Excellent IDE support. A statically typed language. Huge and mature eco-system. First class testing support and much much more.
