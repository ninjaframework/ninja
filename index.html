Ninja web framework

Large web applications in Java made easy.



Happy business. Happy customers.

Ninja will let you finish your next project faster with stunning high quality. Your customers will love what they'll get.

Fun for developers.

Ninja provides everything you need to get productive immediately. And notably Ninja is fast. Really fast. Make a change to the code and see the results instantly.





Get started!





Stable technology

Ninja did not reinvent the wheel - instead we are standing on the shoulders of giants. Ninja is built using the most stable technologies in the industry.

Scalability

Ninja uses the share nothing approach. Servers do not share any common session information. That means you can scale Ninja with ease.

Ready to start

Ninja is a full stack. From quick development to fast testing to easy deployment. And by the way - if you want you can use any servlet container to run your Ninja application.



Pure Open source

Ninja is licensed under Apache 2.0 license. That means the codebase will be open forever. It is welcoming pull requests with new features.

Powerful build system

Maven is one of the best build and library management systems out there. And Ninja is built for it. Even better: Automating your builds in Jenkins becomes as easy as telling Jenkins your repository location.

The best Java citizen

Ninja is built with Maven. That also means you can easily import your project with only one click in any IDE. And you get all goodies of Java for free: Excellent libraries, World-class refactoring support, testing at the heart of the framework and a statically typed language.
