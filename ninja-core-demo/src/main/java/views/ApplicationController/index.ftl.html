Ninja web framework

And developing large web applications becomes fun again.

Fun and Productive

Cut out the time you spend waiting for your application to restart, increase your productivity and complete your projects faster.

CoC!

That means convention over configuration! We designed Ninja to be usable in 99% of the cases without any additional configuration. That speeds up development right from the start. And for the last 1% you can of course configure everything yourself. Just in case...

Java only

Ninja makes it easier to use all your favorite IDEs, build systems and libraries from the Java world.

Powerful build system

Maven is one of the best build and library management systems out there. And Ninja is built for it. Even better: Automating your builds in Jenkins becomes as easy as specifying the pom.

Ready to start

Download the package and you're done. Start hacking with the integrated development server. Push to production with a single command.

Servlets galore!

Servlet containers are the standard in the Java world. Ninja works out of the box with all popular servlet containers. Therefore you get both: Standard compliance and ultra-fast development cycles.
